What is Quick Profile?

Quick Profile is a website that allows users to have a web presence. Users are able to create a personal or professional profile of their choice. They are given a selection of topics ranging from a basic summary of themselves to creating a gallery of images. We have a staff dedicated around the clock to answer any question the user has through email or phone. The important goal that we want from this website is for people to connect with others.

How will Quick Profile help you?

Quick Profile is meant to be a stepping stone for many people who want to present themselves as a professional. The importance of having a web presence gives you an advantage over other people. The purpose of the website to tell people who you are, and what you do. The website can also be a starting point for online businesses. The main objective that we have for the website is for people to create relationships with people who have similar interests.

Future Updates

Quick Profile is still a growing website that can improve with future updates. A future update will allow the user to create a resume and place it in their profile. The update will allow the user to increase the number of images in the gallery from the limited four to six in the profile subpage. Additionlly, the user will able to add new section to the user profile that is related to traveling experiences. Ulimately, the update will improve the responsiveness of the website for all devices.
